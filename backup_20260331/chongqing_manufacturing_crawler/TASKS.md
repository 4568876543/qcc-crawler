```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug

# 2. 在Chrome中登录企查查

# 3. 运行爬虫（修改代码只爬取缺失行业）
python3 run_connect.py
```
