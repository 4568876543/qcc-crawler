# 重庆市制造业企业数据爬虫

## 项目简介

本项目使用 Playwright 爬取企查查网站上重庆市制造业各细分行业在各区县的企业数量，并将数据保存为 Excel 表格。

## 功能特点

- 自动登录企查查网站
- 筛选重庆市制造业企业
- 从页面动态获取最新的区县和制造业细分行业分类
- 获取各细分行业在各区县的企业数量
- 数据保存为 Excel 表格，包含：
  - 总览表：所有区县和行业的企业数量统计
  - 区县工作表：每个区县一个工作表，包含该区县各行业的企业数量
  - 区县汇总表：按区县汇总的企业总数
  - 行业汇总表：按行业汇总的企业总数
- 支持断点续爬
- 反爬策略：随机延迟、异常处理

## 项目结构

```
chongqing_manufacturing_crawler/
```

## 安装依赖

```bash
pip install playwright pandas openpyxl
playwright install
```

## 使用方法

1. 修改 `config.py` 中的配置信息（如需要）
2. 运行 `run.py` 脚本：
   ```bash
   python run.py
   ```
3. 按照提示输入验证码
4. 等待爬虫完成，结果将保存在 `data/chongqing_manufacturing_companies.xlsx`

## 配置说明

在 `config.py` 中可以配置以下信息：

- `PHONE_NUMBER`: 登录手机号
- `HEADLESS`: 是否无头模式运行（默认为 False，建议设置为 False 以便查看爬取过程）
- `SLOWMO`: 操作延迟，单位毫秒（默认为 100）
- `TIMEOUT`: 超时时间，单位毫秒（默认为 30000）
- `PAGE_SIZE`: 每页显示的企业数量（默认为 40）
- `OUTPUT_FILE`: 输出的 Excel 文件路径
- `TEMP_DATA_FILE`: 临时数据文件路径（用于断点续爬）
- `RANDOM_DELAY_MIN` 和 `RANDOM_DELAY_MAX`: 随机延迟范围，单位秒
- `MAX_RETRIES`: 最大重试次数

## 注意事项

1. 本爬虫需要登录企查查账号，请确保手机号能够接收验证码
2. 由于企查查网站可能会更新，爬虫可能需要相应调整
3. 爬取过程中请不要关闭浏览器窗口
4. 如果爬取过程中断，可以重新运行脚本，会从上次中断的地方继续爬取
5. 请合理使用爬虫，遵守网站的使用条款和 robots.txt 规则

## 数据说明

输出的 Excel 文件包含：

1. **总览表**：包含所有区县和行业的企业数量统计
2. **区县工作表**：每个区县一个工作表，包含该区县各行业的企业数量
3. **区县汇总表**：按区县汇总的企业总数
4. **行业汇总表**：按行业汇总的企业总数

数据格式：
- 区县：重庆市的区县名称（从企查查页面动态获取）
- 行业代码：国标行业代码（从企查查页面动态获取）
- 行业名称：行业名称（从企查查页面动态获取）
- 企业数量：该区县该行业的企业数量

## 故障排除

1. **验证码问题**：如果无法收到验证码，请检查手机号是否正确
2. **登录失败**：可能是企查查网站更新了登录流程，请尝试手动登录后再运行爬虫
3. **数据获取失败**：可能是企查查网站更新了页面结构，请检查爬虫代码是否需要调整
4. **Excel文件无法打开**：可能是文件被占用，请关闭所有打开的Excel文件后再运行爬虫
